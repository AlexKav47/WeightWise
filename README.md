# WeightWise

WeightWise is an AI-powered wellness companion web application designed to support users with weight management, healthier habits, meal planning, and fitness guidance. The project focuses on creating a simple and user-friendly experience where users can receive personalised recommendations based on their goals, lifestyle, and input.

This project was built as a modern web application using Next.js, TypeScript, React, Tailwind CSS, and AI-focused tooling.

## Project Overview

The aim of WeightWise is to make weight management easier to understand and more approachable. Instead of only tracking numbers, the application is designed to provide helpful guidance around food choices, exercise, and general wellness.

The app can be developed further into a full wellness platform with features such as progress tracking, personalised plans, goal setting, AI recommendations, and visual dashboards.

## Features
